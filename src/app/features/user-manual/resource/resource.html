Project Folder Structure – L1M2

The application follows a feature-based and scalable folder structure, separating core components, layouts, and feature modules. 

src/
 └── app/
     ├── core/
     │   ├── components/
     │   │   ├── header/
     │   │   └── footer/
     │   └── services/
     │
     ├── layouts/
     │   ├── auth-layout/
     │   └── main-layout/
     │
     ├── features/
     │   ├── auth/
     │   │   └── login/
     │   ├── dashboard/
     │   └── user-manual/
     │
     ├── app-routing.module.ts
     ├── app.component.ts
     └── app.config.ts
  • Core: Shared components like Header & Footer
  • Layouts: Separate layouts for authentication and main app
  • Features: Business modules such as Login, Dashboard, User Manual
  • Routing: Centralized route configuration
